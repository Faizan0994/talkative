# Talkative

A real-time messaging app for seamless communication.

## Features

- Real-time one-on-one messaging
- User authentication and profiles
- User search
- Message history
- Read receipts
- Notifications

## Tech Stack

- **Frontend:** React
- **Backend:** Node.js
- **Database:** PostgreSQL

Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.
